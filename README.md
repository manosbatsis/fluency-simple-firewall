# fluency-simple-firewall ![build](https://github.com/manosbatsis/fluency-simple-firewall/actions/workflows/ci.yml/badge.svg)

## Prerequisites

- [NodeJS](https://nodejs.org/)

## Installation

```
npm install
```

## Usage

Start Ganache
```
npx ganache-cli
```

Run Tests

```
npx truffle test
```
